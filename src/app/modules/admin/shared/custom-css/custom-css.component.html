Editor de CSS Personalizado

Para sobrescribir estilos, use !important cuando sea necesario.

Ejemplo: color: red !important;

Puede utilizar las siguientes propiedades CSS:

  • color
  • font-size
  • font-family
  • background-color
  • border

Además, puede usar las siguientes clases de la aplicación:

  • .content-navbar - Barra de navegación
  • .content-worky - Contenido principal de la página
  • .content-social-app - Área social de la aplicación
