# i18n System — socialNetworkFrontend-app

Custom static translation system. Not ngx-translate — do not add `TranslateService` from that library.

## Files

| File | Purpose |
|---|---|
| `src/translations/translations.ts` | Manager, Proxy, language detection (defaults to Spanish) |
| `src/translations/translations.en.ts` | English strings (~75 KB) |
| `src/translations/translations.es.ts` | Spanish strings (~79 KB) |
| `src/translations/dynamic-translations.en.ts` | Runtime-generated English strings |
| `src/translations/dynamic-translations.es.ts` | Runtime-generated Spanish strings |

## Usage in Templates

### Basic key

```html
{{ 'key.subkey' | workyTranslations }}
```

### Pluralization

```html
{{ 'key' | workyTranslations: { type: 'quantity', count: n } }}
```

### Interpolation with arguments

```html
{{ 'key' | workyTranslations: { type: 'arguments', args: [value1, value2] } }}
```

### Runtime-generated strings (TypeScript)

```typescript
import { getDynamicTranslation } from '@translations/translations';

const label = getDynamicTranslation(someText);
```

## Key Namespaces

| Namespace | Covers |
|---|---|
| `login.*` | Login screen |
| `register.*` | Registration flow |
| `auth.*` | Auth errors and messages |
| `resetPassword.*` | Password reset flow |
| `input.*` | Form input labels and placeholders |
| `button.*` | Button labels |
| `navbar.*` | Navigation labels |
| `publishing.*` | Post creation UI |
| `addPublication.*` | Publication composer |
| `publicationsView.*` | Feed and publication detail |
| `messages.*` | Messaging list and inbox |
| `messageChat.*` | Chat conversation view |
| `admin.*` | Admin panel |
| `APP.UPDATE.*` | Forced update dialog |

## Rules

- Every user-visible string must go through the `workyTranslations` pipe — no hardcoded text in templates.
- When adding a new key, add it to **both** `translations.en.ts` and `translations.es.ts` in the same commit. They must stay in sync at all times.
- Use `getDynamicTranslation()` only for strings that are built at runtime (e.g. server-provided category names).
- Do not import or use `TranslateService` from ngx-translate.
